Proyecto E-commerce (Backend)

Admin API

Estado

Servidor Django en ejecución. Usa el panel Admin para crear productos o la API para consultar endpoints.

Guía rápida

  1. Registrar usuario: POST /api/auth/register/
  2. Obtener token: POST /api/auth/login/
  3. Listar productos: GET /api/products/
  4. Crear pedido (autenticado): POST /api/orders/

Entorno de desarrollo — no use en producción.
